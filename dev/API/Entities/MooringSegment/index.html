MooringSegment

The MooringSegment module represents individual segments of a mooring line, containing the physical properties and finite element discretization.

Types

Mooring.MooringSegments.MooringSegment â€” Type

MooringSegment Struct

This struct is used to define a segment in the mooring system. It includes the following fields:

  • tag::String: Identifier for the segment
  • trian::Triangulation: Triangulation of the segment
  • points::Vector{MooringPoint}: End points of the segment, defined as MooringPoint types.
  • map::Function: Function to map the segment from reference configuration

to undeformed configuration.

  • material::Material: Material properties of the segment
  • density::Real: Submerged density of the segment (default is 1.0)
  • area::Real: Effective cross-sectional area of the segment (default is 1.0)
  • phys_dim::Int: Physical dimension of the segment
