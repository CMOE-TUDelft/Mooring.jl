# MooringPoint

The `MooringPoint` module manages individual points in the mooring system,
such as anchors, fairleads, and connection points.

## Types

```@docs
Mooring.MooringPoints.MooringPoint
```

## Functions

```@docs
Mooring.MooringPoints.get_triangulation
```

Additional functions:
- `get_motion`: Returns the motion type of the point
- `get_boundary_triangulation`: Returns the boundary triangulation
- `get_measures`: Returns measure for integration
